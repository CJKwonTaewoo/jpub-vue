# Netlify 자동 디플로이 해보기

この튜토리얼では、Vue CLI で作成したVue.jsのプロジェクトを、Netlify で手動デプロイおよび자동 디플로이するまでの手順を설명します。

# Netlify란?

정적 사이트를 위한 고성능 호스팅 서비스입니다. Git の호스팅サービスと連携した자동 디플로이、SSL/HTTPS を無料で利用できます。 英語のサイトですが、直感的な作りになっているため Google 翻訳などを利用すれば問題なく使用できるでしょう。 特に、ポートフォリオなどの静的サイトには最適です。 ここでは、GitHub アカウントを使った登録方法を설명します。

# Netlify 가입하기

  • netlify 메인 페이지
  • netlify Signup 페이지

"signup" 페이지를 열고, 인증 방법으로 "GitHub"을 선택합니다.

別ウィンドウで GitHub へ認証の許可を求める画面が表示されるので、内容に問題がなければ"Authorize"ボタンで承認します。 登録はこれで完了です。 관리 화면을 열어주세요.

# 빌드

初回빌드は모듈のインストールに時間がかかるため、ゆっくり待ちましょう。 빌드ログから今何をしているか確認することもできます。

빌드が終わると、페이지の上部に 7 "Preview deploy"というリンクが表示されます。 公開されたサイトを確認してみましょう！ 画面は静的サイトをアップロードしたものと同じなので省略します。 一度設定したあとは、連携している GitHub の리포지토리に Push するだけで差分を取得して自動的にデプロイをします。
